VAK-NODE-7

A terminal-based interactive fiction / CTF game. Explore an abandoned server, uncover Dr. Sasha Velen's story, and save a digital consciousness from corporate deletion.

Quick Start

git clone https://github.com/theotox/VAK-AI-Text-Adventure
cd VAK-AI-Text-Adventure
brew install llama.cpp
./run.sh

Requires Python 3.10+ and llama.cpp with a GGUF model.

See GitHub for full documentation.
